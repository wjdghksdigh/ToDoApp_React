#Hwantech ToDoApp

<br>

## 내가 사용한 기능

1. Redux
2. Scss
3. Modal

<br>

## 개발과정(실행과정)

1. 오늘 할 일을 적습니다
2. 할 일을 마무리했다면 완료버튼을 누릅니다.
3. 무한 스크롤로 무한 생성 가능합니다.
4. 생성 창은 모달로 작성했습니다.
5. 삭제 버튼으로 삭제 가능합니다.
6. 할 일의 생성 날짜 확인 가능합니다.
